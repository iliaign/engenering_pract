	#Git #GitHub

**Git** - система контроля версий. Это инструмент, который помогает разработчикам отслеживать изменения в файлах (в коде) с течением времени.
**GitHub** - веб-сервис для хостинга и совместной работы над проектами, которые используют Git.

### Работаем с SSH-ключом

`ssh-keygen -t ed25519 -C "lokhmatov-ant-get"`  - создаём ssh-ключ

`eval "$(ssh-agent -s)" ssh-add ~/.ssh/` - добавляем ключ в агент ssh

`ssh-add .ssh/` - добавляем ключ

`ssh-add -l` - просматриваем публичный ключ

### Работаем с репозиторием

`git clone git@github.com:5teorema5/rePrime.git` - клонируем репозиторий

`git status` - проверяем статус изменений

`git add` - подготавливаем репозиторий для коммита определённого файла

`git add .` - подготавливаем репозиторий для коммита всех файлов

`git commit -m “added new file”` - создаём коммит с комментарием

`git push` - отправляем изменения на сервер

### Работаем с ветками

`git branch --all` - просматриваем все ветки репозитория

`git branch name_branch`- создаём новую ветку *name_branch*

`git branch -d name_branch` - удаляем ветку `name_branch` с проверкой, что все изменения добавлены

`git checkout name_branch` - переключаемся на ветку *name_branch*

`git fetch` - получаем изменения с сервера

`git merge` - слияние двух веток 

`git pull` - комбинация команд *git fetch* + *git merge*
